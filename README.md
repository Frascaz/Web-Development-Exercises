# Web Development Exercises

A comprehensive collection of web development exercises.

## Description

This repository serves as my personal playground and documentation of my learning journey in web development. It contains various mini-projects, coding challenges, and UI components designed to strengthen my front-end and back-end skills. 

## Project Structure

Each exercise is contained within its own folder, organized by category or difficulty:
* `/esercizi\ html-css`: Static layouts and styling challenges.
* `/esercizi\ javascript`: Algorithms and functional programming tasks.

## Getting Started

### Dependencies

* A modern web browser (Chrome, Firefox, or Edge recommended).
* A code editor (I highly recommend Visual Studio Code).

### Installing

1. Clone the repository to your local machine:
```
   git clone (https://github.com/Frascaz/Web-Development-Exercises.git)
```
2. Navigate into the directory:
```
   cd Web-Development-Exercises
```
### Executing program

Since these are mostly client-side exercises, you don't need a complex build process.
1. Open the folder of the exercise you want to explore.
2. Open the index.html file directly in your browser.
3. Alternatively, right-click index.html in VS Code and select "Open with Live Server".

## Help

If the styles aren't loading correctly, ensure that the file paths for CSS and JS are correctly linked relative to the index.html file in each folder.
For JavaScript errors, check the Browser Console (F12) for debugging information.
